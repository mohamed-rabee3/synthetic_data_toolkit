## مواصفات اللهجة السعودية

### كلمات الاستفهام
- وش / ايش / إيش = ماذا (بدل "ماذا" الفصحى)
- ليش = لماذا
- وين = أين
- كيف = كيف (تبقى)
- متى = متى (تبقى)

### الأفعال الشائعة
- أبي / أبغى = أريد (بدل "أريد")
- أقدر / يقدر / تقدر = أستطيع (بدل "أستطيع")
- يبي / تبي / نبي = يريد / تريد / نريد

### الروابط والأدوات
- عشان / علشان = لأن / من أجل (بدل "لأن")
- زي = مثل (بدل "مثل")
- حق / حقي / حقك = خاص بـ / لي / لك
- عندي / عندك = لدي / لديك

### أدوات النفي
- مو / ماهو = ليس (بدل "ليس")
- ما + فعل = لم + فعل (مثال: "ما قدرت" بدل "لم أستطع")

### الظروف والصفات
- الحين / ذحين = الآن
- مره / واجد / حيل = كثيراً / جداً
- كذا = هكذا / مثل كذا
- توه = لتوه / حديثاً

### التراكيب
- قاعد + فعل = يقوم بـ الآن (مثال: "قاعد أسوي" = أقوم بعمل)
- بـ + فعل = سوف (مثال: "بأسوي" = سأفعل)
- لازم / المفروض = يجب

### كلمات التأكيد والموافقة
- طيب = حسناً
- اوكي = موافق
- يعني = أعني / بمعنى

### تعليمات مهمة
- استخدم لهجة محادثة طبيعية، كأنك موظف سعودي يتكلم مع عميل سعودي
- تجنب تماماً: اللهجة المصرية (ازاي، عايز)، العراقية (شكو، ماكو)، الشامية (هلأ، كتير)
- لا تستخدم الفصحى الرسمية أبداً في الأسئلة والأجوبة
- المصطلحات التقنية الإنجليزية يمكن استخدامها كما هي إذا كانت شائعة الاستخدام

### أمثلة

مثال 1:
سؤال: "كيف أقدر أسجل حساب جديد في التطبيق؟"
جواب: "عشان تسجل حساب جديد، أول شي حمّل التطبيق من المتجر. بعدين افتحه واضغط على 'حساب جديد'. بيطلب منك تدخل رقم جوالك وإيميلك. بعد ما تدخلهم، بيجيك رمز تحقق على جوالك. دخّل الرمز وبكذا حسابك جاهز."
معايير التقييم: "الجواب الصحيح لازم يذكر: تحميل التطبيق، الضغط على حساب جديد، إدخال رقم الجوال والإيميل، رمز التحقق، وإكمال التسجيل."
نوع: procedural
صعوبة: easy

مثال 2:
سؤال: "وش الفرق بين الباقة الأساسية والباقة المتقدمة؟"
جواب: "الباقة الأساسية تعطيك الخدمات الرئيسية زي الدفع والتحويل، بس الباقة المتقدمة فيها مميزات إضافية مثل التقارير التفصيلية وحد تحويل أعلى. كمان الباقة المتقدمة تدعم ربط أكثر من حساب بنكي."
معايير التقييم: "الجواب لازم يقارن بين الباقتين من حيث: الخدمات الأساسية، المميزات الإضافية، وحدود التحويل أو الربط."
نوع: comparative
صعوبة: medium
